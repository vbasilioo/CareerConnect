# CareerConnect
O seu software que conecta profissionais e oportunidades de emprego!


### Contribuindo com o projeto: 

<p>1. Realize um <b>Fork</b> do projeto</p>
<p>2. Crie um branch com a nova feature <b>(git checkout -b staging)</b></p>
<p>3. Realize o Commit <b>(git commit -m 'add COISASADICIONADAS')</b></p>
<p>4. Realize o Push no Branch <b>(git push origin staging)</b></p>
<p>5. Abra um <b>Pull Request</b></p>
